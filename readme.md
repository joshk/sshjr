sshjr
=========

A JRuby library wrapping the Java [SSHJ Jars](https://github.com/shikhar/sshj)

This gem also adds similar functionality to [net-ssh-shell](https://github.com/mitchellh/net-ssh-shell)

Installation
------------

In your Gemfile:

```ruby
gem 'sshjr', :github => 'joshk/sshjr'
```
